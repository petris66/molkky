# Mölkky tulostaulu 🎯

Yksinkertainen Mölkky-pelin kirjanpito ja live-tulostaulu, joka toimii suoraan selaimessa (myös iPhonessa).

---

## 🎮 Käyttö

### Kirjaaja (pelin syöttö)
Avaa tämä linkki:

https://petris66.github.io/molkky/?game=OMA_PELI&v=32

👉 Vaihda `OMA_PELI` haluamaksesi (esim. `vappu2026`)

---

### Katsojat (live-tulostaulu)

Jaa tämä linkki muille:

https://petris66.github.io/molkky/?game=OMA_PELI&view=score&v=32

👉 Sama `game`-nimi kuin kirjaajalla

---

## 📝 Näin käytät

1. Avaa kirjaajalinkki
2. Syötä pelaajat
3. Valitse tavoitepisteet (esim. 50)
4. Paina **Aloita peli**
5. Syötä heitot (esim. `Jari 2`)
6. Jaa katsojalinkki muille

---

## 🔊 Puheohjaus

Voit käyttää puhetta iPhonessa:

Esimerkkejä:
- `Niina 10`
- `Jari 2`
- `huti`

👉 Huom: numerot kannattaa sanoa numeroina (esim. “kaksi” tai “2”)

## 📱 Vinkki (iPhone)

Lisää peli kotinäyttöön:

1. Avaa Safari
2. Paina **Jaa**
3. Valitse **Lisää Koti-valikkoon**

👉 toimii kuin oma appi
